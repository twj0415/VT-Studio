## 0. 快速理解

```txt
一句话：生产页资产节点展示原始资产和衍生资产，支持衍生资产删除和图片更新。
为什么现在做：生产时经常需要基于角色/场景/道具生成某个镜头里的变化版本。
做完后有什么用：用户能看到每个父资产下面有哪些衍生资产，以及它们的图和状态。
这一步不碰什么：不做资产中心普通 CRUD，不做分镜图生成，不做视频生成。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow assets 节点和 production/assets 路由。

只做：
  1. 展示父资产和 derive 子资产
  2. 衍生资产卡片状态
  3. 点击衍生资产进入图片工作流
  4. 保存衍生资产图片
  5. 删除衍生资产
  6. 轮询衍生资产图片生成状态

不做：
  1. 普通资产新增编辑
  2. 分镜面板管理
  3. 视频工作台
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/production/node/assets.vue` | 父资产左侧，衍生资产右侧列表，点击打开 editImage |
| `Toonflow-app/src/routes/production/assets/updateAssetsUrl.ts` | 保存衍生图，插入 o_image 并更新 o_assets.flowId/imageId |
| `Toonflow-app/src/routes/production/assets/deleteAssetsDireve.ts` | 删除衍生资产，删除 flow 和关系 |
| `Toonflow-app/src/routes/production/assets/batchGenerateAssetsImage.ts` | 批量生成衍生资产图片 |
| `Toonflow-app/src/routes/production/assets/pollingImage.ts` | 轮询衍生资产图片状态 |

参考项目关键事实：

```txt
1. 衍生资产是 o_assets.assetsId 指向父资产。
2. deleteAssetsDireve 拼写错误，返回文案也是“视频删除成功”，VT Studio 不能照搬。
3. 删除衍生资产没有删除 o_image 和本地图片文件，VT Studio 必须补。
4. 生成衍生资产图片使用父资产图片作为参考，读取衍生视觉手册。
```

## 3. 用户操作

```txt
入口：
  生产流程图 -> 资产节点。

按钮/操作：
  1. 查看父资产和衍生资产。
  2. 点击衍生资产打开图片工作流。
  3. 删除衍生资产。
  4. 等待衍生资产图片生成完成。

弹窗/表单：
  图片工作流编辑弹窗；删除确认框。

成功反馈：
  图片更新或衍生资产删除成功。

失败反馈：
  生成失败显示原因；删除失败提示。
```

## 4. 要做什么功能

### 1. 展示衍生资产

怎么做：
- 输入：flowData.assets。
- 输出：父资产和 derive 子资产卡片。
- 写什么数据：不写。
- 状态怎么变：生成中/失败/已完成状态展示。
- 异常怎么处理：图片缺失显示缺失状态。
- 限制：只展示当前剧本关联资产及衍生资产。

### 2. 保存衍生资产图片

怎么做：
- 输入：deriveAssetId、imageUrl/filePath、flowId。
- 输出：更新后的衍生资产主图。
- 写什么数据：写 media/image 记录，更新 asset current_media_id 和 flowId。
- 状态怎么变：未生成/失败 -> succeeded。
- 异常怎么处理：文件写入失败或资产不存在提示。
- 限制：文件路径走受控服务。

### 3. 删除衍生资产

怎么做：
- 输入：deriveAssetId。
- 输出：删除结果。
- 写什么数据：删除衍生资产、图片、图片工作流、分镜资产关系和文件。
- 状态怎么变：从 derive 列表移除。
- 异常怎么处理：事务失败回滚。
- 限制：不能删除父资产。

### 4. 轮询衍生图片状态

怎么做：
- 输入：生成中的 deriveAssetIds。
- 输出：状态变化结果。
- 写什么数据：不写。
- 状态怎么变：running -> succeeded/failed。
- 异常怎么处理：轮询失败保留下次。
- 限制：轮询不重复生成。
