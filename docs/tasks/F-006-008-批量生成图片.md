状态：等待用户确认

## 0. 快速理解

```txt
一句话：选中有 prompt 的角色/场景/道具后，批量生成资产图片。
为什么现在做：资产库需要可视化主图，后续分镜和视频会引用这些图。
做完后有什么用：多个资产可以同时进入图片生成，完成后自动显示结果。
这一步不碰什么：不做单资产历史图选择，不处理 clip/audio，不做视频生成。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow `/assetsGenerate/batchGenerateImageAssets`。

只做：
  1. 批量生成图片入口
  2. 选择图片模型和分辨率
  3. 校验 prompt
  4. 写图片占位记录
  5. 调用图片模型
  6. 保存图片文件
  7. 轮询图片状态

不做：
  1. 单资产生成弹窗
  2. 历史图片选择
  3. ComfyUI workflow 设计
  4. 视频生成
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/assets/index.vue` | 批量生成 -> 生成图片，选择模型和分辨率 |
| `Toonflow-app/src/routes/assetsGenerate/batchGenerateImageAssets.ts` | 插入 o_image 占位，更新 o_assets.imageId，并发调用图片模型 |
| `Toonflow-app/src/routes/assets/pollingImageAssets.ts` | 轮询 image state 不为生成中的资产 |
| `Toonflow-app/src/utils/ai.ts` | 图片模型 run 内部处理任务记录 |

参考项目关键事实：

```txt
1. 只支持 role/tool/scene。
2. 分辨率选项 1K/2K/4K。
3. 选中资产没有 prompt 会跳过并提示；全部没 prompt 时终止。
4. 插入图片占位 state=生成中，更新资产 imageId。
5. 成功写 filePath/model/resolution/state=已完成。
6. 失败写 state=生成失败、errorReason。
7. 资产图生成 aspectRatio 固定 16:9，源码未找到可配置比例。
```

## 3. 用户操作

```txt
入口：
  资产中心 role/scene/prop tab -> 勾选资产 -> 批量生成 -> 生成图片。

按钮/操作：
  1. 勾选资产。
  2. 选择图片模型。
  3. 选择分辨率。
  4. 点击开始生成。

弹窗/表单：
  批量生成图片确认弹窗。

成功反馈：
  资产图片状态变生成中，完成后显示图片。

失败反馈：
  未选中、未选模型、未选分辨率、缺 prompt、模型失败。
```

## 4. 要做什么功能

### 1. 校验批量生成参数

怎么做：
- 输入：assetIds、model、resolution。
- 输出：可生成资产列表。
- 写什么数据：不写。
- 状态怎么变：无。
- 异常怎么处理：缺参数提示；没有 prompt 的资产列出并跳过。
- 限制：只允许 role/scene/prop。

### 2. 创建图片占位

怎么做：
- 输入：可生成资产列表。
- 输出：media/image 记录。
- 写什么数据：写 media_files 或 asset_images，状态 running；更新 asset current_media_id。
- 状态怎么变：asset image_status=running。
- 异常怎么处理：占位写入失败时不调用模型。
- 限制：事务写入。

### 3. 调用图片模型并保存

怎么做：
- 输入：prompt、model、resolution、referenceList 可选。
- 输出：图片文件。
- 写什么数据：写项目业务文件，更新 media 状态和 filePath。
- 状态怎么变：running -> succeeded/failed。
- 异常怎么处理：供应商失败写 error_reason。
- 限制：普通图片协议和 ComfyUI 协议由 Model Adapter 区分。

### 4. 轮询图片结果

怎么做：
- 输入：running assetIds。
- 输出：非 running 状态。
- 写什么数据：不写。
- 状态怎么变：页面更新。
- 异常怎么处理：轮询失败保留下一轮。
- 限制：轮询不重复执行生成。

```txt
  media_files.file_path
  media_files.status
  media_files.error_reason
  media_files.model

  asset.pollImageStatus

  更新资产当前图片

任务状态：

轮询/Socket：

模型调用：
  ImageGenerationRequest -> Model Adapter

删除影响：
  无
```

## 7. 偏差

```txt
和 Toonflow 不同的地方：
  内部状态用英文枚举；不同图片协议由 adapter 区分。

原因：
  避免中文状态混乱，支持后续 ComfyUI/RunningHub 等协议扩展。

是否写入 04：
  状态和任务中心接入写 04。
```
