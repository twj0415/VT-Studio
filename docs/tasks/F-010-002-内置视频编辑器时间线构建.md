# F-010-002 内置视频编辑器时间线构建

状态：等待用户确认  
所属菜单：M-010 导出  
对应功能文档：`docs/features/M-010-导出.md`  
原则：先确认本文档，再改代码

## 0. 快速理解

```txt
一句话：构建内置视频编辑器的前端时间线，支持把素材拖到轨道并形成 clip。
为什么现在做：参考项目有 WebAV 编辑器，但它没有后端保存闭环，必须先把边界写清。
做完后有什么用：用户能在内置编辑器里预览和调整素材时间线。
这一步不碰什么：不保存正式剪映导出时间线，不生成剪映草稿，不合成 mp4。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow editVideo 内置视频编辑器的时间线构建能力。

只做：
  1. 素材库数据接入
  2. 拖拽素材到轨道
  3. 自动找可用轨道或创建轨道
  4. 生成 clip
  5. 加载视频缩略图、音频波形、图片缩略图
  6. 切割、删除、撤销、重做
  7. 转场插入规则

不做：
  1. WebAV mp4 导出
  2. 剪映草稿导出
  3. 后端时间线保存
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/production/components/workbench/editVideo/index.vue` | 编辑器主体、工具栏、drop-media、历史快照 |
| `.../mediaLibrary.vue` | 素材库和拖拽数据 |
| `.../videoPreview.vue` | 预览画布 |
| `.../propertyPanel.vue` | 属性面板 |
| `.../utils/trackHelper.ts` | 轨道查找、clip 创建 |
| `.../utils/transitionHelper.ts` | 转场插入规则 |
| `.../utils/mediaLoader.ts` | 缩略图和波形加载 |

参考项目关键事实：

```txt
1. vue-clip-track 使用前端状态管理轨道和 clips。
2. 轨道数量限制：video 5、image 3、audio 3、subtitle 2、text 2、sticker 2、filter 1、effect 2。
3. 默认时长：video/image 5s、audio 30s、subtitle/text/sticker/filter/effect 3s。
4. 转场必须放在同一轨道相邻两个非 transition clip 之间。
5. operationButtons 有 import，但模板注释掉了入口。
6. 源码未找到时间线保存到后端的接口。
```

## 3. 用户操作

```txt
入口：
  视频工作台 -> 视频编辑器模式。

按钮/操作：
  1. 从素材库拖拽素材到轨道。
  2. 添加转场。
  3. 切割选中 clip。
  4. 删除选中 clip。
  5. 撤销、重做。
  6. 调整属性面板。

弹窗/表单：
  属性面板；转场提示。

成功反馈：
  时间线出现 clip，预览能播放。

失败反馈：
  无可用轨道、转场位置不合法、素材加载失败。
```

## 4. 要做什么功能

### 1. 构建素材库

怎么做：
- 输入：initialVideoItems、initialMediaItems、initialImageItems、initialAudioItems。
- 输出：素材库 tabs。
- 写什么数据：不写。
- 状态怎么变：素材加载中 -> 可拖拽。
- 异常怎么处理：缩略图/波形失败时用默认图。
- 限制：素材 URL 必须受控。

### 2. 拖拽生成 clip

怎么做：
- 输入：media item、drop position、trackId 可选。
- 输出：timeline clip。
- 写什么数据：前端时间线状态。
- 状态怎么变：tracks/clips 更新并写历史快照。
- 异常怎么处理：找不到轨道且无法创建时提示。
- 限制：轨道数量不能超过限制。

### 3. 转场插入

怎么做：
- 输入：transition item、drop position。
- 输出：transition clip。
- 写什么数据：前端时间线状态。
- 状态怎么变：相邻 clip 之间插入转场。
- 异常怎么处理：没有相邻 clip 或已有转场时 warning。
- 限制：转场默认 1 秒，类型必须在支持列表内。

### 4. 时间线编辑工具

怎么做：
- 输入：选中 clips 和当前时间。
- 输出：切割、删除、撤销、重做结果。
- 写什么数据：前端时间线状态。
- 状态怎么变：history 更新。
- 异常怎么处理：无选中 clip 时按钮禁用。
- 限制：本任务不保存到后端。
