## 0. 快速理解

```txt
一句话：角景页展示当前项目的角色、场景、道具资产，并显示图片、提示词和已绑定音频。
为什么现在做：音频绑定和批量生成都要先有可操作的角景资产列表。
做完后有什么用：用户能在一个页面看到哪些资产有图、有 prompt、有绑定音频。
这一步不碰什么：不上传新音频，不生成 TTS，不做生产工作台。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow `/cornerScape/getAllAssets` 的角景资产看板。

只做：
  1. 查询 role/scene/tool 父资产
  2. 排除 clip/audio 和子资产
  3. 展示当前图片和历史图片
  4. 展示 prompt、promptState、imageState
  5. 展示已绑定音频
  6. 类型筛选和快捷选择

不做：
  1. 音频上传
  2. TTS 生成
  3. 生产分镜
  4. 资产中心 CRUD
```

```txt
入口：
  项目内顶部菜单 -> 角景。
```

## 4. 要做什么功能

### 1. 查询角景资产

怎么做：
- 输入：projectId、types 可选。
- 输出：role/scene/prop 父资产列表。
- 写什么数据：不写。
- 状态怎么变：loading -> loaded。
- 异常怎么处理：查询失败显示错误状态。
- 限制：不返回 clip/audio，不返回子资产。

### 2. 展示图片和历史图

怎么做：
- 输入：asset current image、historyImages。
- 输出：卡片图片和抽屉历史图。
- 写什么数据：不写。
- 状态怎么变：根据 image status 显示生成中/失败/已完成。
- 异常怎么处理：文件缺失显示缺失状态。
- 限制：预览 URL 走媒体服务，不暴露路径。

### 3. 展示已绑定音频

怎么做：
- 输入：asset_audio_links。
- 输出：boundAudios 标签。
- 写什么数据：不写。
- 状态怎么变：无。
- 异常怎么处理：音频资产已删除时显示缺失。
- 限制：正式字段用 boundAudios，不用 relepedAudio。

### 4. 快捷选择

怎么做：
- 输入：当前资产列表。
- 输出：selectedIds。
- 写什么数据：不写。
- 状态怎么变：选中状态更新。
- 异常怎么处理：生成中资产不可选。
- 限制：选择只影响页面操作，不改数据库。
