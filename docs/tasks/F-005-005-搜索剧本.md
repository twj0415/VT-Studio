# F-005-005 搜索剧本

状态：等待用户确认  
所属菜单：M-005 剧本  
对应功能文档：`docs/features/M-005-剧本.md`  
原则：先确认本文档，再改代码

## 0. 快速理解

```txt
一句话：剧本页按名称搜索当前项目剧本，并展示资产关系和提取状态。
为什么现在做：剧本多了以后需要快速找到某一集。
做完后有什么用：用户能按名称筛剧本，看到提取中、失败和关联资产。
这一步不碰什么：不做分页，不做正文全文搜索，不做资产提取。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow `/script/getScrptApi` 的搜索列表能力。

只做：
  1. 剧本名称搜索
  2. 当前项目剧本列表
  3. 返回关联资产
  4. 展示 extractState 和 errorReason
  5. 空搜索返回全部

不做：
  1. 正文全文搜索
  2. 分页
  3. 高级筛选
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/script/index.vue` | 顶部搜索框点击搜索，调用 searchScripts |
| `Toonflow-app/src/routes/script/getScrptApi.ts` | 按 projectId 和 name 模糊查询，返回关联资产、提取状态、失败原因 |

参考项目关键事实：

```txt
1. getScrptApi 是拼写错误，VT Studio 不能保留。
2. 搜索范围是当前项目。
3. 搜索字段确认是剧本名称。
4. 源码未找到分页，返回当前项目匹配剧本。
5. 列表需要带 relatedAssets、extractState、errorReason。
```

## 3. 用户操作

```txt
入口：
  剧本页顶部搜索框。

按钮/操作：
  1. 输入剧本名称关键词。
  2. 点击搜索。
  3. 清空后搜索全部。

弹窗/表单：
  无。

成功反馈：
  剧本卡片列表更新。

失败反馈：
  搜索失败提示；无结果显示空状态。
```

## 4. 要做什么功能

### 1. 查询剧本列表

怎么做：
- 输入：projectId、keyword。
- 输出：剧本列表。
- 写什么数据：不写。
- 状态怎么变：loading=true -> false。
- 异常怎么处理：失败提示并保留旧列表或显示错误空态。
- 限制：只查当前项目。

### 2. 按名称搜索

怎么做：
- 输入：keyword。
- 输出：name like keyword 的剧本。
- 写什么数据：不写。
- 状态怎么变：搜索后更新列表。
- 异常怎么处理：空关键词查全部。
- 限制：不搜索正文，除非后续单独任务确认。

### 3. 返回关联资产和提取状态

怎么做：
- 输入：scriptIds。
- 输出：relatedAssets、extractStatus、extractError。
- 写什么数据：不写。
- 状态怎么变：卡片 tag 更新。
- 异常怎么处理：资产缺失时显示已删除或缺失状态。
- 限制：状态内部用英文枚举，UI 显示中文。
