## 0. 快速理解

```txt
一句话：资产中心做角色 tab，管理角色资产的列表、新增、编辑、删除、子资产和图片状态。
为什么现在做：角色是剧本资产提取、角景绑定、生产分镜都会复用的核心资产。
做完后有什么用：用户能维护角色名称、描述、提示词、主图和衍生子资产。
这一步不碰什么：不做音频绑定，不生成视频，不做生产衍生资产流程。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow `type=role` 的资产管理能力。

只做：
  1. 角色 tab
  2. 角色父子资产列表
  3. 新增角色
  4. 编辑角色
  5. 删除/批量删除角色
  6. 展示主图、提示词、描述、备注、创建时间
  7. 展示提示词和图片生成状态

不做：
  1. 批量生成提示词
  2. 批量生成图片
  3. 单资产生成图片弹窗
  4. 音频绑定
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/assets/index.vue` | role tab 列表、展开子资产、选择、编辑、删除、生成入口 |
| `Toonflow-web/src/views/assets/components/addAssets.vue` | 新增/编辑普通资产，字段 name/describe/remark/prompt |
| `Toonflow-app/src/routes/assets/getAssetsApi.ts` | 查询父资产和子资产，left join 当前 image |
| `Toonflow-app/src/routes/assets/addAssets.ts` | 新增 `type=role` 资产 |
| `Toonflow-app/src/routes/assets/updateAssets.ts` | 更新 name/describe/remark/prompt |
| `Toonflow-app/src/routes/assets/delAssets.ts` | 删除单个资产 |
| `Toonflow-app/src/routes/assets/batchDelete.ts` | 批量删除，但源码只删 o_assets，有风险 |

参考项目关键事实：

```txt
1. 角色正式参考 type 是 role。
2. 父资产 assetsId 为空，子资产 assetsId 指向父资产 id。
3. 角色图形式是主图 imageId + 历史图片列表 o_image，源码未找到三视图/方位图字段。
4. 生成中资产不能选择、删除、生成。
5. 批量删除源码不删文件、图片、子资产和关系，VT Studio 必须修正。
```

## 3. 用户操作

```txt
入口：
  项目内顶部菜单 -> 资产 -> 角色 tab。

按钮/操作：
  1. 点击新增角色。
  2. 编辑角色。
  3. 删除角色。
  4. 批量删除角色。
  5. 展开查看子资产。
  6. 勾选角色用于后续批量生成。

弹窗/表单：
  新增/编辑角色弹窗；删除确认框。

成功反馈：
  保存/删除成功，刷新角色列表。

失败反馈：
  必填缺失、生成中禁止操作、删除失败、文件清理失败。
```

## 4. 要做什么功能

### 1. 查询角色列表

怎么做：
- 输入：projectId、name、page、pageSize。
- 输出：角色父资产和子资产树。
- 写什么数据：不写。
- 状态怎么变：loading -> loaded。
- 异常怎么处理：查询失败显示错误状态。
- 限制：只查 type=role 且当前项目。

### 2. 新增角色

怎么做：
- 输入：name、describe、remark、prompt。
- 输出：新角色资产。
- 写什么数据：写 assets。
- 状态怎么变：保存中 -> 刷新列表。
- 异常怎么处理：name/describe 缺失不能保存。
- 限制：新增普通角色不选择父资产。

### 3. 编辑角色

怎么做：
- 输入：assetId、name、describe、remark、prompt。
- 输出：更新后的角色。
- 写什么数据：更新 assets。
- 状态怎么变：保存中 -> 刷新列表。
- 异常怎么处理：生成中禁止编辑或明确提示。
- 限制：不能把 role 改成其他类型。

### 4. 删除角色

怎么做：
- 输入：assetIds。
- 输出：删除结果。
- 写什么数据：删除资产、子资产、图片记录、相关关系和可删除文件。
- 状态怎么变：删除中 -> 刷新列表。
- 异常怎么处理：生成中禁止删除；事务失败回滚。
- 限制：批量删除必须复用完整级联删除能力，不能照搬只删 o_assets。

## 7. 偏差

```txt

  1. 批量删除执行完整级联。
  2. 不把三视图/方位图当作已有功能。

  参考项目没有三视图字段；删除源码存在明显残留文件和关系风险。

  批量删除修正写入 04。
```

## 8. 验收

```txt
1. 角色 tab 能查询当前项目角色。
2. 可新增角色。
3. 可编辑角色。
4. 可删除角色。
5. 可批量删除角色。
6. 生成中角色不能被删除。
7. 子资产能展开查看。
8. 删除会清理子资产、图片、关系和文件。
9. 页面不直接访问 SQLite 或文件系统。
```
