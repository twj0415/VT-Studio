## 0. 快速理解

```txt

为什么现在做：生产工作台是主链路页面，用户需要看到内容从剧本到视频的流程。

```

| 参考文件 | 关键逻辑 |

参考项目关键事实：

```txt
1. 主链：script -> scriptPlan -> storyboardTable -> storyboard -> workbench。
2. assets 从 script 分支出来。
3. poster 节点有注释代码，但当前页面未启用。
4. workbench 是前端工作台字段，不在 Production Agent flowData schema 里。
5. 节点只展示当前 flowData，不负责直接读数据库。
```
