# F-005-003 编辑剧本

状态：等待用户确认  
所属菜单：M-005 剧本  
对应功能文档：`docs/features/M-005-剧本.md`

## 0. 快速理解

```txt
一句话：点击剧本卡片打开编辑弹窗，修改名称、内容和关联资产。
为什么现在做：剧本内容和资产关系会影响后续资产提取与生产。
做完后有什么用：用户能修正剧本，并维护它关联的角色/场景/道具。
这一步不碰什么：不删除剧本，不批量新增，不导出 zip。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow `editScript.vue` 和 `/script/updateScript`，并修正 assets 为空不清旧关系的问题。

只做：
  1. 点击卡片打开编辑
  2. 编辑名称和内容
  3. 字数上限校验
  4. 编辑关联资产
  5. 保存剧本和资产关系

不做：
  1. 删除剧本
  2. 资产提取
  3. 生产数据重建
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/script/index.vue` | 点击卡片打开详情编辑弹窗，勾选框和删除按钮 stop |
| `Toonflow-web/src/views/script/components/editScript.vue` | 编辑名称、内容、关联资产，字数限制 |
| `Toonflow-app/src/routes/script/updateScript.ts` | 更新 `o_script`，如果 assets 非空则重建关系 |

参考项目关键事实：

```txt
1. 编辑弹窗名称 maxlength=10。
2. 内容超过 scriptEpisodeLength 时保存按钮禁用。
3. 关联资产类型限制 role/tool/scene。
4. 源码里 assets 为空时不会删除旧关系，VT Studio 必须修正。
5. 点击卡片是编辑；勾选框和删除图标必须 stop，避免误打开编辑。
```

## 3. 用户操作

```txt
入口：
  剧本页 -> 点击剧本卡片。

按钮/操作：
  1. 修改剧本名称。
  2. 修改剧本内容。
  3. 添加或移除关联资产。
  4. 保存。

弹窗/表单：
  编辑剧本弹窗。

成功反馈：
  保存成功，关闭弹窗，刷新剧本列表。

失败反馈：
  名称为空、内容为空、字数超限、保存失败。
```

## 4. 要做什么功能

### 1. 打开编辑弹窗

怎么做：
- 输入：scriptId。
- 输出：剧本表单和已关联资产。
- 写什么数据：不写。
- 状态怎么变：dialogOpen=true。
- 异常怎么处理：剧本不存在提示并刷新列表。
- 限制：勾选框和删除按钮点击不能打开编辑。

### 2. 保存剧本内容

怎么做：
- 输入：name、content。
- 输出：更新后的剧本。
- 写什么数据：更新 scripts。
- 状态怎么变：保存中 -> 成功刷新。
- 异常怎么处理：事务失败回滚。
- 限制：内容变更后要提示已生成生产数据可能失效。

### 3. 保存资产关系

怎么做：
- 输入：selectedAssets。
- 输出：新的剧本资产关系。
- 写什么数据：先删除旧 script_asset_links，再写当前选择。
- 状态怎么变：关系完全同步。
- 异常怎么处理：资产不存在则保存失败或过滤并提示。
- 限制：assets 为空也要删除旧关系。

## 7. 偏差

```txt
和 Toonflow 不同的地方：
  assets 为空时也会清空旧关系。

原因：
  编辑结果必须以当前表单为准，不能保留旧脏关系。

是否写入 04：
  写入，属于修正源码风险。
```

## 8. 验收

```txt
1. 点击卡片能打开编辑弹窗。
2. 勾选框点击不打开编辑。
3. 删除图标点击不打开编辑。
4. 名称和内容必填。
5. 字数超过上限不能保存。
6. 资产关系保存后完全同步。
7. 清空资产后旧关系被删除。
8. 页面不直接访问 SQLite。
```
