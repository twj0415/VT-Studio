# F-006-004 片段素材管理

状态：等待用户确认  
所属菜单：M-006 资产中心  
对应功能文档：`docs/features/M-006-资产中心.md`

## 0. 快速理解

```txt
一句话：资产中心做片段 tab，上传和管理图片、视频、音频类素材片段。
为什么现在做：生产工作台和视频编辑会用到这些通用素材。
做完后有什么用：用户能把本地素材放进项目资产库，并能预览和删除。
这一步不碰什么：不做图片生成，不做 TTS，不做固定 Toonflow 片尾。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow `type=clip` 的素材上传和管理，同时去掉旧品牌固定片尾。

只做：
  1. 片段 tab
  2. 上传图片/视频/音频素材
  3. 编辑名称、描述、备注
  4. 预览素材
  5. 删除/批量删除片段

不做：
  1. 图片生成
  2. 音频生成
  3. 视频剪辑
  4. Toonflow 固定片尾
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/assets/index.vue` | clip tab 上传和列表展示 |
| `Toonflow-app/src/routes/assets/uploadClip.ts` | base64 写入 `/{projectId}/assets/{uuid}.{ext}`，写 o_assets 和 o_image |
| `Toonflow-app/src/routes/assets/getMaterialData.ts` | 查询素材并固定追加 Toonflow 片尾 |
| `Toonflow-app/src/routes/assets/delAssets.ts` | 删除素材 |

参考项目关键事实：

```txt
1. 片段素材可以是图片、视频、音频，不只是视频片段。
2. 前端 accept 只有 .png,.jpg,.jpeg,.mp3,.mp4，但后端支持更多 MIME。
3. 上传后写 o_assets type=clip 和 o_image 媒体记录。
4. getMaterialData 会固定追加 Toonflow片尾 和 /ending.mp4，VT Studio 不能保留。
```

## 3. 用户操作

```txt
入口：
  资产中心 -> 片段 tab。

按钮/操作：
  1. 点击新增片段。
  2. 选择图片/视频/音频文件。
  3. 编辑名称、描述、备注。
  4. 预览素材。
  5. 删除或批量删除。

弹窗/表单：
  文件选择；编辑弹窗；删除确认。

成功反馈：
  上传/保存/删除成功，刷新列表。

失败反馈：
  文件类型不支持、文件过大、写入失败、删除失败。
```

## 4. 要做什么功能

### 1. 上传片段素材

怎么做：
- 输入：本地文件。
- 输出：clip 资产和媒体记录。
- 写什么数据：写 assets、media_files；写项目业务文件。
- 状态怎么变：上传中 -> 成功刷新。
- 异常怎么处理：MIME 不支持、文件过大、写文件失败要提示。
- 限制：renderer 不直接写文件，走 main 文件服务。

### 2. 编辑片段信息

怎么做：
- 输入：assetId、name、description、remark。
- 输出：更新后的片段。
- 写什么数据：更新 assets。
- 状态怎么变：保存中 -> 刷新列表。
- 异常怎么处理：资产不存在或保存失败提示。
- 限制：不改变文件本体。

### 3. 删除片段

怎么做：
- 输入：clip assetIds。
- 输出：删除结果。
- 写什么数据：删除 assets、media_files、文件和引用关系。
- 状态怎么变：删除中 -> 列表刷新。
- 异常怎么处理：被生产工作台引用时提示影响或阻止。
- 限制：不能只删数据库不删文件。

### 4. 去除固定片尾

怎么做：
- 输入：素材列表查询。
- 输出：只返回用户项目素材。
- 写什么数据：不写。
- 状态怎么变：无。
- 异常怎么处理：无。
- 限制：不保留 Toonflow 品牌素材。
