# F-007-002 资产音频绑定

状态：等待用户确认  
所属菜单：M-007 角景音频绑定  
对应功能文档：`docs/features/M-007-角景音频绑定.md`  
原则：先确认本文档，再改代码

## 0. 快速理解

```txt
一句话：单个角色/场景/道具可以选择一个已有音频父资产作为音色参考。
为什么现在做：后续视频或配音需要知道这个资产匹配哪个音色。
做完后有什么用：用户能手动给资产绑定或移除音频。
这一步不碰什么：不生成音频，不上传音频，不绑定多个音色。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow `/cornerScape/updateAssetsAudio` 单个绑定能力。

只做：
  1. 单资产选择音频
  2. 只允许选 audio 父资产
  3. 单选绑定
  4. 移除绑定
  5. 更新卡片和抽屉标签

不做：
  1. 多音色绑定
  2. TTS 生成
  3. 批量 AI 匹配
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/cornerScape/index.vue` | 抽屉里选择音频，调用 openAssetsSelector types=["audio"] multiple=false |
| `Toonflow-app/src/routes/cornerScape/updateAssetsAudio.ts` | 删除旧关系，audioIds 有值则插入一条；超过一个返回错误 |
| `Toonflow-web/src/utils/assetsCheck.ts` | 资产选择器支持类型限制和单选 |

参考项目关键事实：

```txt
1. 选择音频只允许 audio 类型。
2. 后端明确限制仅可绑定一个音色。
3. 删除音频 tag 时调用同一接口，不传 audioIds 即清空关系。
4. o_assetsRole2Audio 字段名写 role，但实际 role/scene/tool 都可绑定。
```

## 3. 用户操作

```txt
入口：
  角景页 -> 点击资产卡片 -> 右侧抽屉 -> 选择音频。

按钮/操作：
  1. 点击选择音频。
  2. 在资产选择器里选一个音色。
  3. 确认。
  4. 点击音频标签关闭按钮移除绑定。

弹窗/表单：
  资产选择器。

成功反馈：
  抽屉和卡片显示已绑定音频。

失败反馈：
  没有音频素材、选择多个、绑定失败。
```

## 4. 要做什么功能

### 1. 选择音频资产

怎么做：
- 输入：assetId。
- 输出：可选 audio 父资产列表。
- 写什么数据：确认前不写。
- 状态怎么变：选择器打开。
- 异常怎么处理：没有音频资产时提示去资产中心添加。
- 限制：只允许单选 audio 父资产。

### 2. 保存绑定关系

怎么做：
- 输入：assetId、audioAssetId。
- 输出：绑定结果。
- 写什么数据：删除旧关系，插入新关系。
- 状态怎么变：boundAudios 更新。
- 异常怎么处理：资产不存在、音频不存在、跨项目绑定失败。
- 限制：一个资产只允许绑定一个音色。

### 3. 移除绑定

怎么做：
- 输入：assetId。
- 输出：解绑结果。
- 写什么数据：删除 asset_audio_links。
- 状态怎么变：boundAudios 清空。
- 异常怎么处理：删除失败提示。
- 限制：不删除音频资产本身。

## 7. 偏差

```txt
和 Toonflow 不同的地方：
  关系字段用 assetId/audioAssetId，不用 assetsRoleId。

原因：
  绑定对象不只是角色，场景和道具也可绑定。

是否写入 04：
  写入，属于命名修正。
```
