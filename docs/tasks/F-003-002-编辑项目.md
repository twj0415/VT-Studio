状态：等待用户确认

## 0. 快速理解

```txt
一句话：项目卡片点编辑，回填项目信息，允许修改名称、简介、模型、画质、比例和手册。
为什么现在做：模型或手册可能失效，打开项目失败时也需要通过编辑项目修复。
做完后有什么用：用户能修正项目配置，避免项目因为模型被删或禁用而无法继续。
这一步不碰什么：不删除项目，不移动项目文件，不编辑小说/剧本/资产。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow editProject 流程，实现项目配置编辑和失效配置修复。

只做：
  1. 项目卡片编辑按钮
  2. 回填项目表单
  3. 回显视频模型 mode
  4. 修改项目字段
  5. 保存编辑
  6. 保存后刷新列表和当前项目上下文
  7. 模型/手册失效提示

不做：
  1. 删除项目
  2. 打开项目跳转
  3. 编辑手册文件内容
  4. 批量编辑项目
```

## 3. 用户操作

```txt
入口：
  项目页 -> 项目卡片 -> 编辑按钮。

按钮/操作：
  1. 点击编辑。
  2. 修改字段。
  3. 重新选择模型或手册。
  4. 点击确认保存。

弹窗/表单：
  项目弹窗编辑模式。

成功反馈：
  编辑成功，关闭弹窗，刷新项目列表。

失败反馈：
  模型失效、手册失效、字段缺失、保存失败。
```

## 4. 要做什么功能

### 1. 打开编辑弹窗

怎么做：
- 输入：项目 id。
- 输出：回填后的项目表单。
- 写什么数据：不写。
- 状态怎么变：dialogShow=true，editProjectData=项目数据。
- 异常怎么处理：项目不存在提示并刷新列表。
- 限制：点击编辑按钮必须阻止卡片打开项目事件。

### 2. 回填字段

怎么做：
- 输入：项目记录。
- 输出：表单字段。
- 写什么数据：不写。
- 状态怎么变：表单进入编辑模式。
- 异常怎么处理：旧字段缺失时显示失效状态。
- 限制：回填必须包含模型、画质、比例、视觉手册、导演手册。

### 3. 回显视频模式

怎么做：
- 输入：videoModel。
- 输出：mode 选项和当前 mode。
- 写什么数据：不写。
- 状态怎么变：视频模式下拉可用。
- 异常怎么处理：模型不存在或供应商禁用时显示失效并要求重新选择。
- 限制：不能因为 mode 拉取失败就静默清空旧值。

### 4. 保存编辑

怎么做：
- 输入：完整表单。
- 输出：更新成功。
- 写什么数据：更新 projects。
- 状态怎么变：保存中 -> 成功后关闭弹窗。
- 异常怎么处理：字段非法、引用失效、数据库失败返回明确 msg。
- 限制：不改变 projectId 和项目业务目录。

### 5. 同步当前项目上下文

怎么做：
- 输入：保存后的项目。
- 输出：当前项目 store 更新。
- 写什么数据：本地当前项目上下文。
- 状态怎么变：如果编辑的是当前打开项目，同步新配置。
- 异常怎么处理：同步失败提示刷新项目。
- 限制：未打开项目则只刷新列表。
