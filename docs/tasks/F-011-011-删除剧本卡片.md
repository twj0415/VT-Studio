# F-011-011 删除剧本卡片

状态：等待用户确认  
所属菜单：M-011 剧本 Agent  
对应功能文档：`docs/features/M-011-剧本Agent.md`  
原则：先确认本文档，再改代码

## 0. 快速理解

```txt
一句话：右侧剧本卡片支持删除，并走剧本模块的删除链路。
为什么现在做：Agent 可能生成多余剧本，用户需要清理。
做完后有什么用：删除不会只从页面消失，而是同步清理真实剧本数据。
这一步不碰什么：不做项目删除，不手写级联逻辑绕过 M-005。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow 剧本卡片删除，同时复用 M-005 剧本删除规则。

只做：
  1. 删除按钮
  2. 二次确认
  3. 有 id 的剧本调用删除服务
  4. 无 id 的临时剧本只从工作区移除
  5. 删除后刷新 planData

不做：
  1. 批量删除
  2. 删除项目
  3. 删除原文
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/scriptAgent/index.vue` | 删除卡片弹确认；有 id 调 `/script/delScript`；无 id 只从 planData 删除 |
| `Toonflow-app/src/routes/script/delScript.ts` | 删除剧本、Agent 工作数据、分镜、视频、关系 |

参考项目关键事实：

```txt
1. 删除剧本会影响 o_script、o_agentWorkData、o_storyboard、o_scriptAssets、o_video。
2. 有 id 的剧本不是只从右侧数组移除。
3. 无 id 的临时剧本只存在于 planData，可直接移除。
4. 删除后 setPlanData + getPlanData。
```

```txt
入口：
  剧本 Agent 右侧 -> 剧本卡片 -> 删除按钮。

按钮/操作：
  1. 点击删除。
  2. 确认。

弹窗/表单：
  删除确认框。

成功反馈：
  删除成功，卡片消失。

失败反馈：
  删除失败提示。
```

## 4. 要做什么功能

### 1. 删除真实剧本

怎么做：
- 输入：scriptId。
- 输出：删除结果。
- 写什么数据：走 M-005 的 script.delete 级联删除。
- 状态怎么变：删除中 -> 刷新 planData。
- 异常怎么处理：删除失败显示原因。
- 限制：必须二次确认，不能只改前端数组。

### 2. 删除临时剧本

怎么做：
- 输入：无 id 的剧本临时 key。
- 输出：工作区数组移除。
- 写什么数据：更新 agent_work_data 快照。
- 状态怎么变：删除中 -> 刷新 planData。
- 异常怎么处理：保存工作区失败则恢复。
- 限制：不能误删同名真实剧本。

### 3. 删除后同步工作区

怎么做：
- 输入：删除结果。
- 输出：新的 planData。
- 写什么数据：agent_work_data 必要同步。
- 状态怎么变：右侧卡片刷新。
- 异常怎么处理：剧本删成功但刷新失败时提示刷新。
- 限制：和 scripts 表保持一致。
