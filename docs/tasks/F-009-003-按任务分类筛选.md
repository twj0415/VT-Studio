# F-009-003 按任务分类筛选

状态：等待用户确认  
所属菜单：M-009 任务中心  
对应功能文档：`docs/features/M-009-任务中心.md`  
原则：先确认本文档，再改代码

## 0. 快速理解

```txt
一句话：任务中心增加任务分类筛选，例如角色图生成、视频生成、导出任务。
为什么现在做：任务多起来后，只看全部列表很难定位某类生成问题。
做完后有什么用：用户可以只看某一类任务，快速查失败和进度。
这一步不碰什么：不新增任务分类，不写业务任务，只读取已有任务里的分类。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow `/task/getTaskCategories`，分类从已有任务动态读取。

只做：
  1. 分类筛选下拉
  2. 从任务表读取已有分类
  3. 选择分类后刷新任务列表
  4. 筛选变化后重置第 1 页

不做：
  1. 固定分类字典管理
  2. 新增任务类型
  3. 分类重命名
  4. 业务任务写入
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/task/index.vue` | 页面进入后调用分类接口，前端补“全部” |
| `Toonflow-app/src/routes/task/getTaskCategories.ts` | 从 `o_tasks` groupBy `taskClass`，过滤空值 |
| `Toonflow-app/src/utils/taskRecord.ts` | 任务写入时传入 `taskClass` |

参考项目关键事实：

```txt
1. 分类不是前端固定枚举，而是从任务表动态读取。
2. 没有产生过的分类，下拉里不会出现。
3. 源码里已看到角色图生成、场景图生成、道具图生成、生成图片、工作流图片生成、生成分镜图片、视频生成。
```

## 3. 用户操作

```txt
入口：
  任务中心筛选栏。

按钮/操作：
  选择“全部分类”或某个任务分类。

弹窗/表单：
  无。

成功反馈：
  表格刷新为对应分类任务。

失败反馈：
  分类加载失败时显示错误提示，任务查询失败时显示错误提示。
```

## 4. 要做什么功能

### 1. 查询分类选项

怎么做：
- 输入：可选项目筛选。
- 输出：已有任务分类选项。
- 写什么数据：不写。
- 状态怎么变：分类下拉 loading -> 可选。
- 异常怎么处理：失败时只保留“全部分类”并提示。
- 限制：不写死业务分类。

### 2. 分类筛选任务

怎么做：
- 输入：category。
- 输出：匹配分类的任务列表。
- 写什么数据：不写。
- 状态怎么变：page=1，保留项目和状态筛选。
- 异常怎么处理：分类不存在时返回空列表。
- 限制：分类值必须来自任务表或合法参数校验。

### 3. 过滤空分类

怎么做：
- 输入：任务表 category。
- 输出：非空分类列表。
- 写什么数据：不写。
- 状态怎么变：无。
- 异常怎么处理：无分类时下拉只有全部。
- 限制：空字符串、null、undefined 不作为分类。

```txt
是否写入 04：

```
