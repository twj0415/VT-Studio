# F-003-007 导演手册管理

状态：等待用户确认  
所属菜单：M-003 项目管理  
对应功能文档：`docs/features/M-003-项目管理.md`

## 0. 快速理解

```txt
一句话：管理项目可选的导演手册，包括新增、编辑、删除、预览和选择。
为什么现在做：项目创建必须选导演手册，后续导演规划、分镜表和生产 Agent 会依赖它。
做完后有什么用：用户能维护导演规则手册，并在项目里选择使用。
这一步不碰什么：不做视觉手册，不做生产工作台，不做 Agent 执行。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow 的 DirectorManual 能力，把导演手册作为本地 Markdown 文件集合管理。

只做：
  1. 查询导演手册列表
  2. 新增导演手册
  3. 编辑导演手册
  4. 删除导演手册
  5. 上传/预览封面图
  6. 3 个 Markdown tab 编辑
  7. 项目选择导演手册
  8. 修正参考项目封面删除误用视觉手册函数的问题

不做：
  1. 视觉手册
  2. 生产 Agent 调用
  3. 分镜生成
  4. Skill 市场下载
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/project/components/projectDialog.vue` | 项目弹窗中展示导演手册卡片，支持选择、新增、编辑、删除、预览 |
| `Toonflow-app/src/routes/project/queryDirectorManual.ts` | 查询导演手册 |
| `Toonflow-app/src/routes/project/addDirectorManual.ts` | 新增导演手册 |
| `Toonflow-app/src/routes/project/editDirectorlManual.ts` | 编辑导演手册，源码文件名有拼写问题 |
| `Toonflow-app/src/routes/project/deleteDirectorManual.ts` | 删除导演手册 |

参考项目关键事实：

```txt
1. 导演手册来自 skills/story_skills。
2. 字段包含 name、directorManual、images、data。
3. 编辑时 directorManual 禁用修改。
4. 导演手册有 3 个 Markdown tab：README、导演规划、分镜表。
5. 源码里导演手册封面删除按钮误用 removeVisualManualCover，VT Studio 必须修正。
6. Toonflow 存在 driector_skills 拼写，VT Studio 新写入必须使用 director_skills，旧拼写只做兼容读取。
```

## 3. 用户操作

```txt
入口：
  新建/编辑项目弹窗 -> 导演手册区域。

按钮/操作：
  1. 点击导演手册卡片选择。
  2. 点击新增导演手册。
  3. 点击编辑导演手册。
  4. 点击删除导演手册。
  5. 点击预览封面。
  6. 上传封面图。
  7. 编辑 Markdown tab。

弹窗/表单：
  导演手册编辑弹窗。

成功反馈：
  新增成功、编辑成功、删除成功、选择成功。

失败反馈：
  名称为空、目录名非法、封面缺失、tab 内容缺失、路径越界、被项目引用不能删除。
```

## 4. 要做什么功能

### 1. 查询导演手册列表

怎么做：
- 输入：无。
- 输出：导演手册列表。
- 写什么数据：不写。
- 状态怎么变：项目弹窗显示卡片。
- 异常怎么处理：目录不存在时显示空状态或初始化默认手册。
- 限制：只读取 story/director skills 白名单目录。

### 2. 新增导演手册

怎么做：
- 输入：name、directorManual/path、images、3 个 tab 内容。
- 输出：新增手册。
- 写什么数据：创建手册目录，写 Markdown 文件和封面图。
- 状态怎么变：新增后列表刷新并可选择。
- 异常怎么处理：目录名非法、目录已存在、封面为空、tab 为空时失败。
- 限制：新写入目录使用 `director_skills` 标准命名。

### 3. 编辑导演手册

怎么做：
- 输入：手册 id/path、name、images、tab 内容。
- 输出：保存成功。
- 写什么数据：更新 Markdown 文件和封面。
- 状态怎么变：列表和当前选择刷新。
- 异常怎么处理：文件缺失、写入失败、路径越界时失败。
- 限制：编辑时 directorManual/path 禁止修改。

### 4. 删除导演手册

怎么做：
- 输入：手册 id/path。
- 输出：删除成功。
- 写什么数据：删除手册目录或标记删除。
- 状态怎么变：列表移除。
- 异常怎么处理：被项目引用时阻止或要求先解除引用。
- 限制：递归删除前必须校验目录在导演手册根目录内。

### 5. 预览封面

怎么做：
- 输入：封面路径。
- 输出：media URL。
- 写什么数据：不写。
- 状态怎么变：打开图片预览。
- 异常怎么处理：图片不存在时显示占位。
- 限制：走 CORE-012 媒体服务，不暴露绝对路径。

### 6. 修正封面删除逻辑

怎么做：
- 输入：导演手册封面索引。
- 输出：删除该导演手册封面。
- 写什么数据：更新导演手册 images。
- 状态怎么变：封面列表更新。
- 异常怎么处理：索引不存在失败。
- 限制：不能调用视觉手册 removeVisualManualCover。

## 7. 偏差

```txt
和 Toonflow 不同的地方：
  1. 新写入目录使用 director_skills，不照搬 driector_skills 拼写错误。
  2. 修正导演手册封面删除误用视觉手册函数的问题。
  3. 删除前检查项目引用。

原因：
  拼写错误和误删封面属于参考项目缺陷，不能带进 VT Studio。

是否写入 04：
  不需要，属于缺陷修正。
```
