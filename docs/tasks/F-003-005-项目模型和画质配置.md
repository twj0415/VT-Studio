# F-003-005 项目模型和画质配置

状态：等待用户确认  
所属菜单：M-003 项目管理  
对应功能文档：`docs/features/M-003-项目管理.md`  
原则：先确认本文档，再改代码

## 0. 快速理解

```txt
一句话：在项目弹窗里选择图片模型、视频模型、视频模式、图片质量和视频比例。
为什么现在做：这些配置会被资产生成、分镜、视频生产反复使用，必须在项目层固定。
做完后有什么用：每个项目都有自己的生成模型和基础画面规格。
这一步不碰什么：不调用模型生成图片/视频，不管理供应商 API Key。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow modelSelect 逻辑，把项目模型和画质配置做成稳定可复用能力。

只做：
  1. 图片模型选择
  2. 视频模型选择
  3. 视频模式选择
  4. 图片质量选择
  5. 视频比例选择
  6. 模型能力读取和失效提示

不做：
  1. 模型供应商配置
  2. 模型测试
  3. 生成任务
  4. 提示词映射
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/components/modelSelect.vue` | 按 type 过滤供应商模型，选择后返回模型信息 |
| `Toonflow-web/src/views/project/components/projectDialog.vue` | 选择图片/视频模型；视频模型变化后生成 mode 选项 |
| `Toonflow-app/src/routes/modelSelect/getModelList.ts` | 查询 enable=1 的供应商模型 |
| `Toonflow-app/src/routes/modelSelect/getModelDetail.ts` | 查询模型详情 |
| `Toonflow-app/src/routes/project/getModelDetails.ts` | 项目内查询模型细节 |

参考项目关键事实：

```txt
1. 图片模型只显示 enable=1 供应商里的 type=image。
2. 视频模型只显示 enable=1 供应商里的 type=video。
3. 视频模式来自模型的 mode 字段。
4. 图片质量是 1K/2K/4K。
5. 视频比例是 16:9/9:16。
6. mode 可能是字符串，也可能是数组型多参考模式，源码会转 JSON 字符串保存。
```

```txt
入口：
  新建/编辑项目弹窗。

按钮/操作：
  1. 打开图片模型选择器。
  2. 选择图片模型。
  3. 选择图片质量。
  4. 打开视频模型选择器。
  5. 选择视频模型。

```

## 4. 要做什么功能

### 1. 图片模型选择

怎么做：
- 输入：模型类型 image。
- 输出：启用供应商下的 image 模型列表。
- 写什么数据：确认项目表单时写项目配置。
- 状态怎么变：选中后表单 imageModel 更新。
- 异常怎么处理：无可用模型时提示先去设置模型供应商。
- 限制：不能显示禁用供应商模型。

### 2. 图片质量选择

怎么做：
- 输入：1K/2K/4K。
- 输出：imageQuality。
- 写什么数据：确认项目表单时写项目配置。
- 状态怎么变：表单更新。
- 异常怎么处理：非法值不能提交。
- 限制：图片质量不等同于视频分辨率。

### 3. 视频模型选择

怎么做：
- 输入：模型类型 video。
- 输出：启用供应商下的 video 模型列表。
- 写什么数据：确认项目表单时写项目配置。
- 状态怎么变：选中后清空旧 mode 并加载新 mode 选项。
- 异常怎么处理：无可用模型时提示先配置供应商。
- 限制：不能显示 image/text/tts 模型。

### 4. 视频模式选择

怎么做：
- 输入：选中视频模型的 mode 能力。
- 输出：mode 选项。
- 写什么数据：确认项目表单时写项目配置。
- 状态怎么变：mode 下拉更新。
- 异常怎么处理：mode 为空或不支持时不能提交。
- 限制：数组型多参考模式要序列化为稳定结构，不乱存页面展示文案。

### 5. 视频比例选择

怎么做：
- 输入：16:9 或 9:16。
- 输出：videoRatio。
- 写什么数据：确认项目表单时写项目配置。
- 状态怎么变：表单更新。
- 异常怎么处理：非法值不能提交。
- 限制：比例影响后续分镜/视频，不直接代表图片模型尺寸。

### 6. 模型失效提示

怎么做：
- 输入：项目已保存模型 + 当前供应商模型列表。
- 输出：有效/失效状态。
- 写什么数据：不写。
- 状态怎么变：表单显示红色失效提示。
- 异常怎么处理：无。
- 限制：不能静默替换为其他模型。
