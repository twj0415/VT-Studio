## 0. 快速理解

```txt
一句话：首次打开应用显示欢迎引导，引导用户配置模型、配置 Agent、切换语言，允许跳过。
为什么现在做：模型和 Agent 是后续功能的基础，新用户需要入口，但不能保留 Toonflow 品牌内容。
做完后有什么用：首次使用不会迷路，可以直接进入关键设置。
这一步不碰什么：不做模型配置表单本身，不做 Agent 配置表单本身。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow 欢迎引导流程，但清理 Toonflow Logo、GitHub、二维码等旧痕迹。

只做：
  1. 首次启动显示欢迎引导。
  2. 支持开始配置。
  3. 支持跳过。
  4. 支持切换语言。
  5. 支持打开模型供应商设置。
  6. 支持打开 Agent 设置。
  7. 支持完成后不再显示。

不做：
  1. 不保留 Toonflow Logo。
  2. 不保留 Toonflow GitHub。
  3. 不保留企业微信二维码。
  4. 不做实际模型配置逻辑。
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/components/hello.vue` | `helloGuideDone` 控制首次引导；步骤包含欢迎、模型服务、Agent、完成 |
| `Toonflow-web/src/pages/workbench/index.vue` | 主布局挂载 hello 组件和 setting 弹窗 |
| `Toonflow-web/src/stores/setting.ts` | 设置 `activeMenu` 和 `showSetting` 打开对应设置页 |

参考项目关键事实：

```txt
1. `helloGuideDone=false` 时显示欢迎引导。
2. 跳过和完成都会设置 `helloGuideDone=true`。
3. 配置模型会打开设置弹窗并切到 `vendorConfig`。
4. 配置 Agent 会打开设置弹窗并切到 `agentConfog`。
5. 完成时有 JSConfetti 动画。
6. 欢迎页里有 ToonFlow Logo、Toonflow GitHub、企业微信二维码，这些都不能保留。
```

## 3. 用户操作

```txt
入口：
  首次启动主界面自动弹出。

按钮/操作：
  1. 开始配置。
  2. 跳过。
  3. 切换语言。
  4. 配置模型服务。
  5. 配置 Agent。
  6. 完成。

弹窗/表单：
  欢迎引导弹窗。

成功反馈：
  打开对应设置 tab，或关闭引导并记录已完成。

失败反馈：
  打开设置失败时提示；持久化失败时提示但允许关闭。
```

## 4. 要做什么功能

### 1. 首次显示控制

怎么做：
- 输入：`helloGuideDone`。
- 输出：是否显示欢迎弹窗。
- 写什么数据：读写前端持久化。
- 状态怎么变：false 显示，true 不显示。
- 异常怎么处理：读取失败时默认显示一次。
- 限制：不能每次启动都弹。

### 2. 跳过引导

怎么做：
- 输入：用户点击跳过。
- 输出：关闭引导。
- 写什么数据：`helloGuideDone=true`。
- 状态怎么变：弹窗关闭。
- 异常怎么处理：写入失败时提示。
- 限制：跳过不自动打开设置。

### 3. 配置模型入口

怎么做：
- 输入：点击配置模型。
- 输出：打开设置弹窗，activeMenu=`vendorConfig`。
- 写什么数据：写前端设置状态。
- 状态怎么变：欢迎引导可关闭或保持，设置弹窗打开。
- 异常怎么处理：设置弹窗打开失败时提示。
- 限制：模型供应商具体表单归 F-002-003。

### 4. 配置 Agent 入口

怎么做：
- 输入：点击配置 Agent。
- 输出：打开设置弹窗，activeMenu=`agentConfig`。
- 写什么数据：写前端设置状态。
- 状态怎么变：设置弹窗打开。
- 异常怎么处理：设置弹窗打开失败时提示。
- 限制：正式 key 用 `agentConfig`，不使用 `agentConfog`。

### 5. 语言切换

怎么做：
- 输入：选择语言。
- 输出：i18n 切换。
- 写什么数据：写语言设置。
- 状态怎么变：引导文案即时更新。
- 异常怎么处理：未知语言回退中文。
- 限制：完整语言设置归 F-002-002。

### 6. 完成引导

怎么做：
- 输入：点击完成。
- 输出：关闭引导并记录完成。
- 写什么数据：`helloGuideDone=true`。
- 状态怎么变：弹窗关闭。
- 异常怎么处理：写失败提示。
- 限制：动画可有可无，但不能影响完成状态。
