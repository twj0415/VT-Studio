# F-005-006 导出剧本 zip

状态：等待用户确认  
所属菜单：M-005 剧本  
对应功能文档：`docs/features/M-005-剧本.md`  
原则：先确认本文档，再改代码

## 0. 快速理解

```txt
一句话：选中剧本后导出 zip，每集剧本一个 txt 文件。
为什么现在做：这是剧本文本导出，和后面的剪映草稿导出不是一回事。
做完后有什么用：用户能把剧本文本打包备份或给别人查看。
这一步不碰什么：不导出剪映草稿，不导出视频，不导出分镜图片。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow `/script/exportScript`。

只做：
  1. 选择剧本导出
  2. 每个剧本写一个 txt
  3. 打包 zip
  4. 处理非法文件名、重名、超长文件名
  5. 导出失败提示

不做：
  1. 剪映导出
  2. WebAV mp4 导出
  3. 分镜图片 zip
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/script/index.vue` | 选中剧本后请求 `/script/exportScript`，responseType=blob，下载 `script_YYYY-MM-DD.zip` |
| `Toonflow-app/src/routes/script/exportScript.ts` | 用 zip stream，每个剧本写 `${script.name}.txt` |

参考项目关键事实：

```txt
1. 导出的是剧本文本 zip。
2. 后端响应 Content-Type=application/zip。
3. 后端文件名 scripts.zip，前端下载名 script_YYYY-MM-DD.zip。
4. 源码没处理非法文件名和重名。
5. 没选中剧本时按钮禁用，但仍需要保护提示。
```

## 3. 用户操作

```txt
入口：
  剧本页顶部 -> 导出剧本。

按钮/操作：
  1. 勾选剧本。
  2. 点击导出剧本。
  3. 选择保存位置或直接下载。

弹窗/表单：
  可选保存目录弹窗，按桌面端实现决定。

成功反馈：
  zip 导出成功，显示路径或触发下载。

失败反馈：
  未选中、文件名处理失败、写 zip 失败。
```

## 4. 要做什么功能

### 1. 选择剧本导出

怎么做：
- 输入：scriptIds。
- 输出：待导出剧本列表。
- 写什么数据：不写数据库。
- 状态怎么变：导出中 -> 成功。
- 异常怎么处理：未选择时提示。
- 限制：只能导出当前项目剧本。

### 2. 生成 zip

怎么做：
- 输入：剧本 name/content。
- 输出：zip 文件。
- 写什么数据：写导出文件。
- 状态怎么变：导出中 -> 完成。
- 异常怎么处理：某个剧本写入失败要列出失败项。
- 限制：txt 文件名必须清理非法字符和重名。

### 3. 返回导出结果

怎么做：
- 输入：zip 路径或 blob。
- 输出：导出成功反馈。
- 写什么数据：可选写导出历史，第一版不强制。
- 状态怎么变：按钮 loading 结束。
- 异常怎么处理：失败显示 msg。
- 限制：这不是剪映导出，不进入 M-010。

## 7. 偏差

```txt
和 Toonflow 不同的地方：
  处理非法文件名、重名和超长文件名。

原因：
  Windows 文件系统限制必须处理，否则导出会失败。

是否写入 04：
  不需要，属于健壮性增强。
```

## 8. 验收

```txt
1. 未选中时不能导出。
2. 选中剧本能生成 zip。
3. 每集剧本一个 txt。
4. 重名剧本不会覆盖。
5. 非法文件名能清理。
6. 导出失败有提示。
7. 不和剪映导出混淆。
8. 页面不直接访问 SQLite 或文件系统。
```
