# F-005-007 从剧本提取资产

状态：等待用户确认  
所属菜单：M-005 剧本  
对应功能文档：`docs/features/M-005-剧本.md`  
原则：先确认本文档，再改代码

## 0. 快速理解

```txt
一句话：选中剧本后，用模型从内容里提取角色、场景、道具资产。
为什么现在做：资产中心和生产工作台需要这些资产，不能全靠手动添加。
做完后有什么用：剧本会自动关联已有或新建的资产，并显示提取状态。
这一步不碰什么：不生成资产图片，不做角景音频绑定，不生成分镜。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow `/script/extractAssets` 的异步资产提取。

只做：
  1. 选择剧本提取资产
  2. 写等待/进行中提取状态
  3. 调用文本模型提取结构化资产
  4. 新建资产或复用已有资产
  5. 写剧本资产关系
  6. 失败写失败原因

不做：
  1. 资产图片生成
  2. 音频绑定
  3. 分镜生成
  4. 提取取消
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/script/index.vue` | 选中剧本后点击提取资产，接口返回后清空选中并刷新 |
| `Toonflow-app/src/routes/script/extractAssets.ts` | 先置 extractState=2，后台改 0，调用 universalAi，成功写资产和关系 |
| `Toonflow-web/src/utils/assetsCheck.ts` | 资产类型 role/tool/scene |
| `Toonflow-app/src/lib/fixDB.ts` | 启动时把残留 extractState=0 改失败 |

参考项目关键事实：

```txt
1. HTTP 成功只代表开始提取，不代表提取完成。
2. extractState=2 等待提取，0 正在提取，1 成功，-1 失败。
3. groupSize 来自 otherSetting.assetsBatchGenereateSize。
4. 模型使用 universalAi。
5. 提示词 type=scriptAssetExtraction，优先 useData 再 data。
6. 输出结构包含 newAssets 和 existingAssetRefs。
7. 资产提取没有写 o_tasks，VT Studio 建议写任务中心。
```

## 3. 用户操作

```txt
入口：
  剧本页 -> 勾选剧本 -> 提取资产。

按钮/操作：
  1. 勾选一个或多个剧本。
  2. 点击提取资产。

弹窗/表单：
  可选确认框。

成功反馈：
  提示已开始提取，卡片显示等待/提取中。

失败反馈：
  未选中、模型配置缺失、提示词缺失、提取失败。
```

## 4. 要做什么功能

### 1. 启动资产提取

怎么做：
- 输入：scriptIds、projectId。
- 输出：提取任务启动结果。
- 写什么数据：scripts.extract_status=waiting/running。
- 状态怎么变：idle/failed -> waiting -> running。
- 异常怎么处理：未选中或剧本不存在返回错误。
- 限制：只处理当前项目剧本。

### 2. 调用模型提取资产

怎么做：
- 输入：剧本内容、已有资产列表、提取提示词。
- 输出：newAssets、existingAssetRefs。
- 写什么数据：模型成功后写资产和关系。
- 状态怎么变：running -> succeeded/failed。
- 异常怎么处理：模型失败、空输出、结构非法写失败原因。
- 限制：模型调用走 main Model Adapter。

### 3. 新建或复用资产

怎么做：
- 输入：提取结果。
- 输出：assets 和 script_asset_links。
- 写什么数据：新建 role/scene/tool 资产；复用已有资产；重建本批剧本关系。
- 状态怎么变：剧本关联资产更新。
- 异常怎么处理：重名资产按规则处理，事务失败回滚。
- 限制：existingAssetRefs 不能只按 name 盲目匹配，需处理重名。

### 4. 写任务中心记录

怎么做：
- 输入：提取请求和结果。
- 输出：任务记录。
- 写什么数据：tasks。
- 状态怎么变：running -> succeeded/failed。
- 异常怎么处理：任务写入失败不应吞掉业务失败。
- 限制：这是 VT Studio 增强，需要写 04。


## 7. 偏差

```txt
和 Toonflow 不同的地方：
  1. 建议写入任务中心。
  2. 处理重名资产，不只按 name 精确匹配。

原因：
  资产提取是异步模型任务，失败需要可追踪；重名资产会导致错误关联。

是否写入 04：
  写入。
```
