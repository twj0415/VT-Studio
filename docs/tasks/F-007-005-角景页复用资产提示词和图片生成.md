# F-007-005 角景页复用资产提示词和图片生成

## 0. 快速理解

```txt
一句话：角景页的提示词润色、图片生成、历史图切换必须复用资产中心能力。
为什么现在做：同一类资产生成如果写两套逻辑，状态和文件一定会乱。
做完后有什么用：角景页可以批量生成 prompt、批量生成图、单项重新生成，并和资产中心数据一致。
这一步不碰什么：不重新发明图片生成服务，不做视频生成，不做 ComfyUI 页面。
```

## 1. 本次做什么

```txt
目标：
  对齐 Toonflow 角景页复用 `/assetsGenerate/*` 和 `/assets/saveAssets` 的行为，同时拆清接口语义。

只做：
  1. 单项提示词保存
  2. 单项 AI 润色
  3. 单项重新生成图片
  4. 批量生成提示词
  5. 批量生成图片
  6. 历史图片切换
  7. 取消生成入口规则

不做：
  1. 单独新建一套生成服务
  2. 生产工作台图片工作流
  3. 视频生成
  4. ComfyUI workflow 编辑
```

## 2. 参考项目怎么做

| 参考文件 | 关键逻辑 |
|---|---|
| `Toonflow-web/src/views/cornerScape/index.vue` | 左侧批量生成提示词/图片，右侧抽屉单项润色/生成/历史图切换 |
| `Toonflow-app/src/routes/assetsGenerate/polishAssetsPrompt.ts` | 单项提示词润色 |
| `Toonflow-app/src/routes/assetsGenerate/batchPolishAssetsPrompt.ts` | 批量提示词润色 |
| `Toonflow-app/src/routes/assetsGenerate/generateAssets.ts` | 单项图片生成 |
| `Toonflow-app/src/routes/assetsGenerate/batchGenerateImageAssets.ts` | 批量图片生成 |
| `Toonflow-app/src/routes/assets/saveAssets.ts` | 保存 prompt 或 imageId，但语义过重 |
| `Toonflow-app/src/routes/assetsGenerate/cancelGenerate.ts` | 取消图片生成 |

参考项目关键事实：

```txt
1. 角景页会传 otherTextPrompt，所以不触发资产中心批量提示词缺参冲突。
2. 提示词失焦保存复用 saveAssets，存在误清 imageId 风险。
3. 历史图片再次点击可能传 imageId undefined，存在误清当前图风险。
4. 取消生成传 freshItem.imageId。
5. onUnmounted 重置本地状态不等于取消后端任务。
```

## 3. 用户操作

```txt
入口：
  角景页左侧批量设置；右侧单项配置抽屉。

按钮/操作：
  1. 批量生成提示词。
  2. 批量生成图片。
  3. 抽屉里 AI 润色。
  4. 抽屉里重新生成。
  5. 失焦保存 prompt。
  6. 选择历史图片。
  7. 取消生成。

弹窗/表单：
  右侧抽屉；取消确认框。

成功反馈：
  prompt、图片、历史图状态更新。

失败反馈：
  缺 prompt、缺模型、缺分辨率、模型失败、取消失败。
```

## 4. 要做什么功能

### 1. 提示词保存

怎么做：
- 输入：assetId、prompt。
- 输出：保存结果。
- 写什么数据：更新 assets.prompt。
- 状态怎么变：dirty -> saved。
- 异常怎么处理：保存失败回滚 UI。
- 限制：单独能力 `asset.updatePrompt`，不复用重接口。

### 2. 单项和批量提示词生成

怎么做：
- 输入：assetId 或 assetIds、otherTextPrompt。
- 输出：prompt 更新。
- 写什么数据：写 assets.prompt/status/error。
- 状态怎么变：running -> succeeded/failed。
- 异常怎么处理：失败原因展示。
- 限制：复用 F-006-007 服务。

### 3. 单项和批量图片生成

怎么做：
- 输入：资产、prompt、model、resolution。
- 输出：图片结果。
- 写什么数据：写 media/image 记录和文件。
- 状态怎么变：running -> succeeded/failed。
- 异常怎么处理：缺 prompt 列出资产名称。
- 限制：复用 F-006-008/F-006-009 服务。

### 4. 历史图片切换

怎么做：
- 输入：assetId、mediaId。
- 输出：主图更新。
- 写什么数据：更新 assets.current_media_id。
- 状态怎么变：selectedHistoryId 更新。
- 异常怎么处理：mediaId 为空时不调用保存。
- 限制：不能误清当前图片。

### 5. 取消生成

怎么做：
- 输入：imageId/mediaId。
- 输出：取消结果。
- 写什么数据：更新 media/task 状态。
- 状态怎么变：running -> cancelled。
- 异常怎么处理：无生成中记录提示。
- 限制：不能把 onUnmounted 本地重置当取消。
