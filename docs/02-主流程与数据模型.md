# 主流程与数据模型

## 主数据链路

```text
Project
→ StoryboardItem[]
→ ImageCandidate[]
→ StoryboardItem.selectedImageId
→ VideoSegment[]
→ StoryboardItem.selectedVideoSegmentId
→ CompositionTask.outputPath
→ ExportRecord
```

## 内容创作

输入：

```text
topic | paste | article
视频包引用
画幅
目标时长
目标分镜数
语言
```

输出：

```text
StoryboardItem.sourceText
StoryboardItem.narrationText
```

阻断：

```text
没有已确认段落不能进入分镜
paste 模式不能私自改写原文
AI 失败时必须允许机械切分和手动确认
```

## 分镜

输出字段：

```text
sourceText
narrationText
visualDescription
characterIds / characters
locationId / sceneDescription
imagePrompt
negativePrompt
videoPrompt
durationSeconds
shotSize
cameraMotion
transitionType
lockFlagsJson
selectedImageId
selectedVideoSegmentId
```

规则：

```text
sourceText / narrationText 至少一个非空
visualDescription、imagePrompt、durationSeconds 是生图前置
selectedImageId 只由生图选择写入
selectedVideoSegmentId 只由视频选择写入
lockFlagsJson 是锁定唯一来源
```

## 生图

输入：

```text
StoryboardItem
ImageInputPlan
ProviderModel 或 WorkflowPreset
创作规则
参考图
```

输出：

```text
ImageCandidate[]
StoryboardItem.selectedImageId
```

规则：

```text
required 输入缺失不能生成
远程结果必须下载到受控目录
重生成追加 revision，不覆盖旧候选
用户确认后写 selectedImageId
没有 selectedImageId 不能进入视频
```

## 视频

输入：

```text
StoryboardItem.selectedImageId
ImageCandidate.imagePath
VideoInputPlan
ProviderModel 或 WorkflowPreset
videoPrompt
durationSeconds
```

输出：

```text
VideoSegment[]
StoryboardItem.selectedVideoSegmentId
```

规则：

```text
普通 image_to_video 必须引用 selectedImageId 对应图片
重生成追加 revision，不覆盖旧片段
用户确认后写 selectedVideoSegmentId
没有 selectedVideoSegmentId 不能进入合成
```

## 合成

输入：

```text
StoryboardItem.index
StoryboardItem.selectedVideoSegmentId
VideoSegment.videoPath
FFmpeg / FFprobe sidecar
```

输出：

```text
CompositionTask.outputPath
final.mp4
ExportRecord
```

规则：

```text
所有分镜必须有 selectedVideoSegmentId
视频文件必须存在
ffprobe 必须可读
输出路径必须受 PathGuard 控制
```

## 配置引用链

```text
VideoPack
→ CreativeRule 引用
→ ProviderModel / WorkflowPreset 推荐引用
→ AssetReference 引用
→ ProjectRuntimeConfig
→ TaskStep snapshot
```

禁止把视频包、规则、Provider、模型、workflow 混成一个大 JSON。
