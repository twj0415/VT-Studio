# 模块规格

## 我的作品

目标：查看、创建、继续、归档、恢复、删除作品，并看到进度和失败恢复入口。

区域：

```text
顶部：开始创作、搜索、筛选、排序
主体：作品卡片或列表
右侧：作品详情、任务摘要、输出记录、失败信息
底部：工作区和全局任务摘要
```

操作：

```text
开始创作、打开作品、继续生成、打开输出目录、复制作品、归档、恢复、删除
```

验收：

```text
列表来自 SQLite
生命周期操作真实写回
默认不显示 deleted
删除二次确认
失败作品能跳到任务中心或对应工作台步骤
```

当前代码映射：

```text
页面：src/src/pages/home/index.vue
Store/API：src/src/entities/project
缺口：任务中心跳转目标尚未存在；资源消耗统计仍需确认来源。
```

## 任务中心

目标：全局查看任务，定位失败，重试、取消、继续，查看资源消耗和诊断。

操作：

```text
打开作品、跳到步骤、跳到分镜行、重试失败项、取消任务、继续任务、打开输出目录、导出诊断包
```

验收：

```text
任务列表是全局的
工作台展示当前作品任务摘要
失败能定位步骤和行号
能重试失败项
能取消 running 任务
日志脱敏
不显示虚假资源消耗
```

当前代码映射：

```text
后端命令：list_tasks / get_task_detail / cancel_task / resume_task / retry_task_step
前端底座：src/src/entities/task/api.ts、src/src/entities/task/store.ts
已补：src/src/pages/tasks 页面、/tasks 路由、AppShell Rail 入口、WorkspaceHeader 快捷菜单入口。
当前缺口：任务详情还需要更细的行级失败定位、脱敏日志、Artifact 展示和资源消耗真实统计。
```

## 内容创作

目标：把主题、已有文案或文章变成可确认的分镜段落。

操作：

```text
保存草稿、AI 生成 / 整理、切分预览、编辑段落、合并、拆分、删除、移动、恢复原文、锁定、确认进入分镜
```

状态：

```text
empty_input
too_long
needs_split_preview
splitting
ai_generating
ai_failed
ready_to_confirm
confirmed
draft_saved
```

写回：

```text
Project.inputType
Project.inputProcessMode
Project.sourceText / sourceTextPath
StoryboardItem.sourceText
StoryboardItem.narrationText
```

验收：

```text
paste 不改写原文
topic/article 走 LLM 后仍进入切分预览
长文本有风险提示和切分建议
AI 失败后可手动确认
确认前不写入最终 StoryboardItem
script-editor 不作为必经页
```

当前代码映射：

```text
页面：src/src/pages/create-project/index.vue
当前缺口：需要核对长文本检测、切分预览、AI 失败手动兜底是否真实闭环；script-editor 页面文件仍存在但未注册路由。
```

## 分镜

目标：把已确认段落变成可生图、可视频化的结构化镜头表。

主字段：

```text
sourceText
narrationText
visualDescription
characterIds / characters
locationId / sceneDescription
imagePrompt
negativePrompt
videoPrompt
durationSeconds
lockFlagsJson
```

操作：

```text
生成分镜、重新生成未锁定、确认分镜、进入生图、新增、复制、删除、上移、下移、拖拽排序、拆分、合并、锁定、跳转到指定行、筛选无效行
```

Inspector 可以放：

```text
校验问题、AI 助手、候选图 / 视频预览、高级字段、下游影响提示
```

Inspector 不能放主字段第二套编辑器。

验收：

```text
表格是主编辑入口
锁定字段不会被批量重生成覆盖
删除、拆分、合并提示下游影响
行号跳转可用
进入生图前校验 visualDescription、imagePrompt、durationSeconds
```

当前代码映射：

```text
页面：src/src/pages/storyboard-editor/index.vue
后端命令：get_storyboard / update_storyboard_item / batch_update_storyboard_items / generate_storyboard_with_llm / approve_task_step
已补：行号跳转；分镜“重新生成”走真实 LLM adapter，输出结构化字段并写回 Storyboard；生成失败进入 Task 失败记录。
当前缺口：Inspector 是否只做辅助、删除/拆分/合并下游影响提示、单行 AI 助手需要继续审计和补齐。
```

## 生图

目标：按分镜行生成多张候选图，并选择每行最终图。

操作：

```text
批量生成全部、只生成未生成、只生成失败项、批量重试、编辑提示词、预览最终提示词、单行生成、重新生成、选择候选图、确认图片、查看大图、替换本地图、保存为角色参考、保存为场景参考、保存为风格参考、清理未选候选、撤回本次生成、进入视频
```

状态：

```text
not_generated
input_missing
running
failed
waiting_select
selected
confirmed
```

写回：

```text
ImageCandidate[]
StoryboardItem.selectedImageId
Task / TaskStep / Artifact
```

验收：

```text
选择模型后请求 ImageInputPlan
required 缺失不能生成
最终 prompt 预览真实展示组装来源
重生成追加候选，不覆盖旧候选
选择候选写 selectedImageId
所有行有 selectedImageId 后才能进入视频
```

当前代码映射：

```text
页面：src/src/pages/image-generation/index.vue
Store/API：src/src/entities/scene
后端：scene_service + provider_service
当前缺口：真实外部 Image adapter 未实现；最终 prompt 预览是否来自后端真实组装需要审计。
```

## 视频

目标：用每行最终图生成候选视频片段，并选择每行最终片段。

操作：

```text
批量生成全部、只生成未生成、只生成失败项、批量重试、编辑视频提示词、单行生成、重新生成、预览视频、选择片段、确认片段、查看输入要求、审核失败中性化确认、清理未选片段、进入合成
```

写回：

```text
VideoSegment[]
StoryboardItem.selectedVideoSegmentId
Task / TaskStep / Artifact
```

验收：

```text
没有 selectedImageId 的行不能生成
模型切换后重新请求 VideoInputPlan
重生成追加 VideoSegment
选择片段写 selectedVideoSegmentId
所有行有 selectedVideoSegmentId 后才能进入合成
```

当前代码映射：

```text
页面：src/src/pages/video-generation/index.vue
Store/API：src/src/entities/scene
后端：scene_service + provider_service
已补：Video / Workflow 通过 ProviderManager 调用通用 HTTP 异步轮询 adapter 或 ComfyUI adapter，Provider 输出 bytes 后由 Service 写入受控 workspace。
当前缺口：真实供应商配置样例、参数映射细化、真实 key smoke；审核失败中性化是否真实闭环需要审计。
```

## 合成导出

目标：检查所有视频片段，合成 final.mp4，并导出可交付文件或诊断包。

操作：

```text
检查片段、开始合成、取消合成、重新合成、预览 final.mp4、打开输出目录、导出 final.mp4、导出项目包、导出诊断包、查看错误
```

状态：

```text
not_ready
ready
checking
transcoding
concatenating
succeeded
failed
exporting
exported
```

验收：

```text
所有分镜有 selectedVideoSegmentId 才能合成
ffprobe 检查片段
concat 或转码成功
final.mp4 可播放
导出包不含密钥和绝对路径
诊断包脱敏
```

当前代码映射：

```text
页面：src/src/pages/composition/index.vue
后端：task_service、ffmpeg_service、export_service、diagnostic_service
当前缺口：真实 ffmpeg/ffprobe sidecar smoke、final.mp4 可播放验证尚需实际环境确认。
```


```text
入口：当前受 showAiToolsEntry 控制，可能隐藏。
```

## 创作资源

子页面：

```text
视频包、创作规则、素材库、画风设定、角色设定、场景设定
```

视频包保存默认参数、规则引用、推荐模型 / workflow 引用、素材引用；不保存 API Key、Provider 连接、prompt 正文副本、候选产物、任务日志。

创作规则保存 prompt_body、output_schema、params_schema、测试样例；不直接执行 Provider，不直接写业务表。

素材库只放可复用 Asset。作品产物和候选产物默认留在作品目录，用户显式保存后才进入素材库。

当前代码映射：

```text
页面：src/src/pages/creative-resources/index.vue
后端：video_pack_service、prompt_service、media_service、style_service、character_service、location_service
当前缺口：需要审计删除引用保护、素材预览、保存候选为素材是否真实闭环。
```
