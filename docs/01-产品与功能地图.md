# 产品与功能地图

## 产品定义

桌面端短视频创作工作台：把主题、文案或文章转成可编辑分镜，再生成候选图、候选视频，最后合成并导出成片。

## 目标用户

```text
短视频创作者
内容运营
短剧 / 知识类视频生产者
需要批量产出视频素材的小团队
```

## 第一版必须解决

1. 从主题、文案、文章开始创作。
2. 确认和编辑每条分镜。
3. 每条分镜生成多张候选图。
4. 用户选择最终图。
5. 每条最终图生成候选视频片段。
6. 用户选择最终视频片段。
7. 合成 `final.mp4`。
8. 失败能定位、重试、诊断。

## 不做偏

1. 不做纯聊天工具。
2. 不做模板市场。
3. 不把 AI 工具当主创作入口。
4. 不把小说、数字人、素材成片塞进第一版必经主线。
5. 不复制参考项目代码、UI、素材、品牌和规则正文。
