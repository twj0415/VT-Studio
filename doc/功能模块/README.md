# 功能模块总索引

> 本文件把 `doc/功能模块` 下所有功能模块按当前开发优先级重新整理。
> 这里是“功能模块索引 + 开发阅读入口”，不是替代单个模块文档。
> 真正开发顺序仍以项目 `plan/阶段路线图.md` 的 TODO 为准。

---

## 0. 当前产品主线

当前优先跑通的 workflow：

```text
workflowType = image_to_video
```

主流程：

```text
输入文字
→ 分镜
→ 生图
→ 图生视频
→ 合成
```

页面主线：

```text
新建项目
→ 分镜工作台
→ 生图表格页
→ 图生视频页
→ 合成页
→ 导出 / 打开输出目录
```

关键原则：

```text
1. doc/功能模块 是完整需求库，不等于当前开发顺序。
2. TTS、字幕、封面、BGM 不是不做，而是后续增强 TODO。
3. 数字人口播、素材成片、纯图文成片是独立 workflow，不混进 image_to_video 主线。
4. 生图页必须是表格 / 列表工作台：每行一个分镜。
5. 图生视频读取生图阶段已选图片，不能绕过图片选择。
```

## 7. 模块边界红线

1. 内容导入不负责最终生成，只负责创建项目和归一化输入。
2. 脚本生成不是当前主线必经页面。
3. 分镜阶段不强制景别、运镜、转场。
4. 生图提示词可以并入生图表格页行内。
5. AI 生图必须支持候选图和已选图，不再是单 `imagePath`。
6. 图生视频必须读取已选图。
7. 合成必须优先支持视频片段直接拼接。
8. TTS、字幕、封面、BGM 是 TODO 增强项，不是删除项。
9. 数字人口播、素材成片、纯图文成片是独立 workflow。
10. 所有真实密钥只进系统钥匙串。
