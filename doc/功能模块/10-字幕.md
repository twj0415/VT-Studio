# 字幕

> 这是给 AI 开发用的功能说明书。读完这一篇，就能把“字幕”模块从零写出来，不用再问。
> 配套阅读：分镜字段看 `底层设计/数据结构.md`；模板渲染看 `功能模块/12-模板与动效.md`；TTS 看 `功能模块/09-配音TTS.md`。

## 一、这个功能是什么

字幕模块负责把分镜文案变成适合短视频观看的字幕，并在需要时参与渲染或合成。

字幕要做，但不是当前 `image_to_video` 主线的前置条件。

当前主线可以先无字幕合成：

```text
输入文字 → 分镜 → 生图 → 图生视频 → 合成 final.mp4
```

启用字幕后，它作为增强步骤插入：

```text
分镜文案 / TTS 音频时长 → 字幕断句 → 字幕样式 → 可选烧录 / 可选导出字幕文件
```

字幕包含两件事：

```text
1. 字幕断句：把旁白切成适合屏幕显示的 chunks
2. 字幕样式：字体、描边、位置、安全区、逐字/卡拉OK效果
```

## 子功能 1：自动字幕断句

### 界面

- 每条分镜显示文案和断句结果。
- 支持一键重新断句。
- 每行显示字数提示。

### 系统逻辑

1. 读取 StoryboardItem.narrationText 或 sourceText。
2. 按 contentLanguage 和 aspectRatio 选择断句规则。
3. 中文竖屏默认每行 12~18 字。
4. 优先按标点断句。
5. 标点过少时按语义词组或长度切。
6. 写入 StoryboardItem.subtitleChunks。

### 数据结构

```json
{
  "itemId": "item_001",
  "narrationText": "很多人以为读书只是消遣，其实它在悄悄改变你的判断力",
  "subtitleChunks": [
    "很多人以为读书只是消遣",
    "其实它在悄悄改变你的判断力"
  ]
}
```

### 坑提醒

- 不要按固定字数硬切，会切断语义。
- 不要一行太长，移动端看不清。

## 子功能 2：字幕编辑

### 界面

- 字幕编辑器：每行一个输入框。
- 支持合并上一行、拆成两行、删除空行。
- 实时预览在画面上的位置。

### 系统逻辑

1. 用户编辑 subtitleChunks。
2. 保存到 StoryboardItem。
3. 如果只改字幕断句，不需要重跑 TTS。
4. 如果改了 narrationText 正文，要提示是否同步影响 TTS、视频提示词和后续合成。

### 坑提醒

- `subtitleChunks` 是显示断句，不等于 narrationText 原文。
- 不要因为编辑断句就改旁白音频。

---

## 子功能 3：字幕样式预设

### 界面

字幕样式包含：

- 字体
- 字号
- 字重
- 字色
- 描边颜色
- 描边宽度
- 阴影
- 位置：底部/中部/顶部
- 背景条
- 安全区边距

### 系统逻辑

1. 用户选择字幕样式预设。
2. 保存到项目模板参数或字幕配置。
3. 模板渲染或 FFmpeg 字幕烧录时读取。

### 参考来源

- Pixelle `pixelle_video/services/frame_html.py::HTMLFrameGenerator.generate_frame`。
- Pixelle `templates/1080x1920/*.html`。

### 坑提醒

- 复杂字幕样式默认走 HTML 模板渲染更灵活。
- 简单字幕烧录可以后续接 FFmpeg，但不要和复杂模板混写。

## 子功能 4：逐字 / 卡拉OK字幕

### 系统逻辑

字幕精度分两档：

1. 基础版：按 subtitleChunks 分行显示，无逐字时间。
2. 增强版：根据音频时长和文本长度估算每字时间，生成逐字高亮。

字幕时间结构建议：

```json
{
  "itemId": "item_001",
  "startTime": 0,
  "endTime": 4.82,
  "chunks": [
    { "text": "很多人以为读书只是消遣", "startTime": 0, "endTime": 2.1 },
    { "text": "其实它在悄悄改变你的判断力", "startTime": 2.1, "endTime": 4.82 }
  ]
}
```

### 坑提醒

- 没有字级时间戳时，不要假装精确。
- 按字数比例估算时，文档和代码都标明是 estimated。

---

## 子功能 5：字幕安全区

### 系统逻辑

1. 竖屏 vertical_9_16 默认底部字幕，但避开平台 UI 区域。
2. 封面/标题场景可切换中部字幕。
3. 模板参数里保留 `subtitleSafeBottom / subtitleSafeLeft / subtitleSafeRight`。

### 坑提醒

- 抖音/视频号底部有操作区，字幕不能贴底。

---

## 子功能 6：输出 subtitles.json

### 系统逻辑

1. 遍历 StoryboardItem[]。
2. 根据每条 item.durationSeconds 或 audioDurationSeconds 累加全局时间。
3. 输出 subtitles.json。
4. 模板渲染、最终导出都可读取。

### 输出示例

```json
{
  "contentLanguage": "zh-CN",
  "items": [
    {
      "itemId": "item_001",
      "startTime": 0,
      "endTime": 4.82,
      "text": "很多人以为读书只是消遣，其实它在悄悄改变你的判断力",
      "chunks": ["很多人以为读书只是消遣", "其实它在悄悄改变你的判断力"]
    }
  ]
}
```

## 四、这个模块对外提供什么

写回：

- StoryboardItem.subtitleChunks
- subtitles/subtitles.json
- 字幕样式配置

使用方：

- `12-模板与动效.md` 可用 HTML 渲染字幕。
- `13-视频合成.md` 可选择烧录字幕或导出字幕文件。
- 不启用字幕时，当前主线仍可输出 final.mp4。
