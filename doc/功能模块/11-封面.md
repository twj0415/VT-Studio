# 封面

> 这是给 AI 开发用的功能说明书。读完这一篇，就能把“封面”模块从零写出来，不用再问。
> 配套阅读：模板渲染看 `功能模块/12-模板与动效.md`；导出看 `功能模块/21-导出与项目备份.md`；当前开发顺序看项目 `plan/阶段路线图.md`。

## 一、这个功能是什么

封面模块负责生成短视频封面图。封面不是随便截一帧，而是由标题、主图、模板和样式组合出来。

封面要做，但不是当前 `image_to_video` 主线的前置条件。

当前主线可以先输出：

```text
final.mp4
```

启用封面后，输出：

```text
final.mp4
cover/cover.png
导出元数据
```

核心产物：

```text
coverPath
coverTitle
coverTemplateId
coverSourceItemId
```

## 二、子功能清单

1. 标题生成 / 编辑
2. 选择封面主图
3. 封面模板
4. 生成封面图
5. 手动调整

---

## 子功能 1：标题生成 / 编辑

### 界面

- 标题输入框。
- AI 生成标题按钮。
- 多个标题候选。

### 系统逻辑

1. 如果用户已有标题，直接使用。
2. 如果没有标题，根据主题、输入文字、分镜摘要生成 15 字以内标题。
3. 标题用于封面和历史列表。

### 参考来源

- Pixelle `pixelle_video/utils/content_generators.py::generate_title`。
- Pixelle `pixelle_video/prompts/title_generation.py`。

### 坑提醒

- 标题不能太长，否则封面排版崩。
- 不要生成标题党到偏离原内容。

---

## 子功能 2：选择封面主图

### 界面

- 从已选图片中选择。
- 从视频首帧中选择。
- 支持上传本地图片。
- 支持 AI 重新生成封面主图。

### 系统逻辑

1. 默认推荐第一条或视觉冲击最强的一条 StoryboardItem 已选图。
2. 用户可手动选择。
3. 上传图片要复制进 `cover/` 或 assets 目录。
4. 入库只存相对路径。
5. `coverSourceItemId` 指向来源分镜行。

### 坑提醒

- 不要直接拿外部绝对路径。
- 不要因为改封面重跑整个视频。

---

## 子功能 3：封面模板

### 界面

- 模板卡片：知识科普、情绪文案、故事悬念、清单盘点、小红书风。
- 可调标题颜色、字号、位置、背景遮罩。

### 系统逻辑

1. 模板本质是 HTML。
2. 模板参数走 `{{param:type=default}}`。
3. 渲染时传入 title、image、style 参数。

### 参考来源

- Pixelle `pixelle_video/services/frame_html.py::HTMLFrameGenerator.parse_template_parameters`。
- Pixelle `templates/1080x1920/*.html`。

---

## 子功能 4：生成封面图

### 系统逻辑

1. 读取封面模板。
2. 注入标题、主图、颜色、字体等参数。
3. 用 Playwright sidecar 控制随包 Chromium 截图。
4. 输出 `cover/cover.png`。
5. 写入项目封面路径。

### 参考来源

- Pixelle `pixelle_video/services/frame_html.py::HTMLFrameGenerator.generate_frame`。

### 坑提醒

- 封面渲染要独立于视频片段渲染，不能为了改封面重跑整个视频。
- Playwright / Chromium 路径和模板资源必须遵守安全规范。

---

## 子功能 5：手动调整

### 界面

- 标题拖动位置。
- 字号 / 颜色 / 描边调节。
- 背景模糊 / 暗角 / 遮罩。

### 系统逻辑

1. 用户调整模板参数。
2. 实时预览。
3. 保存参数并重新渲染 cover.png。

---

## 四、这个模块对外提供什么

输出：

```json
{
  "coverPath": "cover/cover.png",
  "coverTitle": "普通人变清醒的开始",
  "coverTemplateId": "knowledge_bold",
  "coverSourceItemId": "item_001"
}
```

使用方：

- 历史列表。
- 导出结果。
- 视频预览页。
- 不启用封面时，当前主线仍可输出 final.mp4。
