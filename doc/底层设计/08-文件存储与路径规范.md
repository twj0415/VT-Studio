# 文件存储与路径规范

> 这篇定义文件目录、FileBucket、相对路径、PathGuard 和导入导出规则。所有文件读写最终仍要遵守 `Provider与安全.md` 的 PathGuard 安全红线。

## 一、核心原则

```text
1. 入库只存相对路径，不存绝对路径。
2. 所有文件都归入 FileBucket。
3. 所有读写必须走 StorageService + PathGuard。
4. 前端不允许把任意系统路径直接交给 FFmpeg / Chromium / Provider 使用。
5. 远程 URL 必须下载转存本地后再入库。
6. 用户导入文件要复制到工作区，不长期引用原始路径。
```

---

## 二、FileBucket

FileBucket 取值以 `01-枚举字典与配置规范.md` 为准：

```text
workspace
projects
assets
templates
outputs
cache
temp
sidecars
logs
```

## 四、路径存储规则

数据库字段只存相对路径：

```text
images/scene_001.png
audio/scene_001.mp3
segments/scene_001.mp4
exports/final.mp4
```

不要存：

```text
C:\Users\xxx\Desktop\a.png
D:\project\workspace\projects\proj_xxx\...
file:///D:/xxx/a.png
https://remote-cdn.com/a.png
```

远程 URL 可以作为临时下载源，但最终入库必须是本地相对路径。

---

## 五、StorageService 规则

业务代码禁止直接：

```rust
std::fs::read(user_path)
std::fs::write(user_path, data)
```

必须：

```rust
storage_service.read_file(bucket, relative_path)
storage_service.write_file(bucket, relative_path, data)
storage_service.copy_imported_file(source_path, target_bucket, target_relative_path)
```

StorageService 内部必须调用 PathGuard。

---

## 六、PathGuard 规则

```text
1. 所有路径先 canonicalize / resolve。
2. 解析真实路径后判断是否在允许根目录内。
3. 禁止用字符串 startsWith 判断边界。
4. 禁止 ../ 或符号链接绕出工作区。
5. 禁止模板渲染联网加载未知资源。
6. sidecar 执行参数里的所有路径都必须来自 SafePath。
```

---

## 七、任务产物隔离

每个任务独立目录：

```text
workspace/projects/proj_xxx/tasks/task_xxx/
```

好处：

```text
便于断点续跑
便于清理临时产物
便于导出历史任务
避免不同任务覆盖文件
```

---

## 八、导入文件规则

用户上传或选择本地文件时：

```text
1. 后端通过文件选择结果拿到原始路径。
2. PathGuard 校验读取权限。
3. 复制到 workspace/assets 或当前 project/task 目录。
4. 入库保存复制后的相对路径。
5. 后续流程只使用工作区内文件。
```

---

## 九、项目存储格式

本地工作区不是单个项目文件，而是：

```text
SQLite 数据库
+ workspace 文件目录
```

规则：

```text
1. SQLite 是项目结构化数据的唯一权威源。
2. workspace 文件目录保存图片、视频、音频、模板、任务产物等文件本体。
3. 数据库只保存相对路径和元数据，不保存文件本体。
4. `project.json` 只允许作为导出快照、诊断快照或项目包辅助文件，不是第二套主数据源。
5. 代码读取项目详情必须以 SQLite 为准，不能优先读 project.json。
6. 导出迁移格式是 project_package.zip，整工作区备份格式是 backup.zip。
```

项目包格式：

```text
project_package.zip
  manifest.json
  db_export.json
  projects/proj_xxx/
  assets/
  templates/user/
```

备份格式：

```text
backup.zip
  app.sqlite
  workspace/projects/
  workspace/assets/
  workspace/templates/user/
  workspace/prompts/user/
```

---

## 十、删除规则

删除文件前必须确认：

```text
1. 文件在允许 bucket 内。
2. 文件确实属于当前项目或任务。
3. 不删除用户原始文件。
4. 删除任务临时文件不能影响历史记录需要的最终产物。
```
